Proximity-Based Multi-Turn Optimization: Practical Credit Assignment for LLM Agent Training

Yangyi Fang, Jiaye Lin, Xiaoliang Fu, Cong Qin, Haolin Shi, Chang Liu, Peilin Zhao.

CCF-A ACL 2026 Oral
Paper Code
TL;DR:Softens cross-turn comparison boundaries by state proximity and corrects gradient strength by intra-group difficulty...

In multi-turn agent RL, turns within a trajectory group carry rich comparable signal, but heterogeneous states make naive grouping mislead credit assignment. We soften step-wise comparison boundaries by state proximity and rescale gradients by intra-group difficulty — two lightweight mechanisms adding only +1.09% overhead.

Placing Puzzle Pieces Where They Matter: A Question Augmentation Framework for Reinforcement Learning

Yangyi Fang, Jiaye Lin, Xiaoliang Fu, Cong Qin, Haolin Shi.

CCF-A ACL 2026
Paper
TL;DR:Bottleneck-driven task augmentation: locates high-impact steps via value attribution and synthesizes a difficulty-adaptive curriculum...

A bottleneck-driven automatic task-augmentation framework: value attribution locates high-impact steps in a task chain, variant tasks are synthesized around the key bottlenecks, and a difficulty-adaptive mechanism produces curriculum training data — mitigating overfitting on easy tasks and reward sparsity on hard ones.

How to Allocate, How to Learn? Dynamic Rollout Allocation and Advantage Modulation for Policy Optimization

Yangyi Fang, Jiaye Lin, Xiaoliang Fu, Cong Qin, Haolin Shi, Chaowen Hu, Lu Pan, Ke Zeng, Xunliang Cai.

CCF-A ACL 2026
Paper Code
TL;DR:Sequence-level variance-minimizing dynamic allocation + token-level gradient-entropy update stabilization...

We derive a variance-minimizing dynamic rollout-allocation policy at the sequence level, and introduce a gradient-entropy-bounded update-magnitude stabilization at the token level. Outperforms 80/20 (Qwen) and Clip-Cov / KL-Cov (Shanghai AI Lab).

From logπ to π: Taming Divergence in Soft Clipping via Bilateral Decoupled Decay of Probability Gradient Weight

Xiaoliang Fu*, Yangyi Fang*, Jiaye Lin*, Chaowen Hu, Cong Qin, Zekai Shao, Binbin Zheng, Lu Pan, Ke Zeng.

CCF-A ACL 2026
Paper Code
TL;DR:Proves IS gradients should align from logπ to π; bilateral decoupled soft clipping cures boundary gradient divergence...

We theoretically show that importance-sampling gradient design should align from logπθ to πθ, and propose bilateral decoupled soft clipping to resolve boundary gradient divergence. Surpasses GPPO / CE-GPPO (Kuaishou) and CISPO (MiniMax).

ANCHOR: Taming Entropy Dynamics for Stable and Efficient Reasoning of Large Language Models

Cong Qin*, Yangyi Fang*, Jiaye Lin*, Xiaoliang Fu*, Chaowen Hu, Huanyao Zhang, Yifu Guo, Zheyuan Gu, Kaitong Qin, Jiawen Kang, Deyang Ding, Peilin Zhao.

CCF-A KDD 2026
TL;DR:Tames entropy-collapse-induced gradient deadlock; +14.4% on LiveCodeBench Avg@32 without verbose generation...

Targets the gradient deadlock caused by entropy collapse in LLM RL. ANCHOR surpasses all baselines across six reasoning benchmarks, with +14.4% Avg@32 on code reasoning (LiveCodeBench) and without inducing verbose generation.
